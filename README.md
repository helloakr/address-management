# 通信录管理系统

## 		项目结构

|    main.go     |   入口文件   |
| :------------: | :----------: |
|     ruoter     |   路由目录   |
|  middlewares   |    中间件    |
|  controllers   |  控制器目录  |
| servicesOption |  服务层目录  |
|     models     |     模型     |
|    database    | 数据库初始化 |
| sessionsOption |  会话初始化  |
